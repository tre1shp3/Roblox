# Functions and what they do.

**Hook:InitializeHooks()** : Place all hooks in the game.
**Hook:RemoveHooks()** : De-initializes all hooks in the game.
**Hook:ChangeObjectValue(Object, Property, Value)** : Changes the property to the set value. Blocks other property writes.
**Hook:RemoveSingular(Object, Property)** : Removes any hooks related to the property of that object.
**Hook:Hook(Object, Property)** : Starts hooking the property of an object.
