# Functions and what they do.

**Main:RBX_Signal_Disable(RBXScriptSignal)** : Disables ALL connections of type on the part TEMPORARILY.

**Main:ChangeSignalStop(Object)** : Disables Changed and discovered GetPropertyChangedSignal connections TEMPORARILY.

**Main:SortAndLogFunctions()** : Sorts all functions found by filtergc.

**Main:LogLoadedModules()** : Logs all required modules and their env.

**Main:FunctionDepthCheck(Function)** : Searches a function in depth and returns data in a single BIG string.
